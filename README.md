# E-Commerce Sales Analytics System

## Project Overview

This project is an end-to-end Data Analytics project developed using Python, MySQL, Pandas, SQL, and Power BI. It generates realistic e-commerce datasets, performs data cleaning and validation, stores data in MySQL, executes advanced SQL queries, generates business reports, and visualizes insights using an interactive Power BI dashboard.

---

## Features

- Generate realistic e-commerce datasets using Python
- Data Cleaning using Pandas
- Data Validation
- MySQL Database Integration
- Advanced SQL Queries
  - Joins
  - Aggregate Functions
  - CTEs
  - Window Functions
  - Cohort Analysis
- Command Line Reporting Tool
- Power BI Interactive Dashboard
- Customer, Revenue and Retention Reports

## Command Line Reporting Tool

Run Customer Report

```bash
python cli_report.py customer
```

Run Revenue Report

```bash
python cli_report.py revenue
```

Run Retention Report

```bash
python cli_report.py retention
```

---

## How to Run

1. Generate datasets
2. Clean and validate data
3. Import data into MySQL
4. Execute schema.sql
5. Execute queries.sql
6. Run analysis.py
7. Run cli_report.py
8. Open Power BI Dashboard
